<h4>You load your website faster than ever</h4>
When your website compresses the HTML code, CSS and JS within the same this can cause a great improvement in loading speed, not only in charge of the HTML file but in general.
It also helps that you can not see your code easily. The plugin is optimized for the Google bot to find faster your links and this causes helps the SEO of your website.

<strong>Here are some of its main features: </strong>
<ul>
	<li>Enable / Disable: Enable / Disable the "iMInify" plugin</li>
	<li>Comments: Disable the elimination of html comments to reduce the code</li>
	<li>CSS: Disable the CSS code reduction in HTML</li>
	<li>Javascript: Disable the Javascript code reduction in HTML</li>
	<li>SEO Optimizer: It comes bundled SEO optimization for your website.</li>
</ul>

The anger plugin improved over time, but now is ready and working. This tested on a website more than 1 million hits a day and work perfectly :)

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Avatar Coquette'
3. Activate 'iMinify' from your Plugins page.
4. Visit 'Setting' > 'iMinify'.
